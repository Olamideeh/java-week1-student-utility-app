Project Description
The Java Student Utility App is a console-based application built to demonstrate basic Java programming concepts such as variables, operators, decision making, loops, methods, and input handling.
The app uses a menu-driven system that allows users to perform simple operations interactively.

This project was developed as a Week 1 coding task to practice Java fundamentals and GitHub workflow.

Features
The application provides the following options:

Add Two Numbers
Accepts two integers from the user
Displays their sum

Check EVEN or ODD
Accepts a number
Uses a ternary operator to determine if the number is EVEN or ODD

Print Numbers from 1 to N
Accepts a number N
Prints numbers from 1 to N using a for loop

Display Day of the Week
Accepts a number between 1 and 7
Displays the corresponding day using a switch statement
Exit
Terminates the program with a thank-you message
To Run the program
Clone this repository https://github.com/Olamideeh/java-week1-student-utility-app/tree/main
Open the folder up in your IDE
Run the programme by click on the play button next to the StudentUtilityApp class.
